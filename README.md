# Maidpromax Management System (Backend)

🚀 A professional-grade RESTful API built with Go, Gin, and GORM.

## ✨ Features
* **Role-Based Access Control (RBAC):** Distinct flows for Admins, Employers, and Nannies.
* **Security:** JWT Authentication and secure Complaint Protocol (Invisible to users).
* **Monitoring:** Admin dashboard for user verification, job assignments, and rating audits.
* **Scalability:** Full pagination on all administrative lists.

## 🛠 Tech Stack
* **Language:** Go 1.2x
* **Framework:** Gin Gonic
* **ORM:** GORM (PostgreSQL)
* **Auth:** JWT (v5)

## 🏁 Getting Started
1. Clone the repo.
2. Setup your `.env` file.
3. Run `go run cmd/main.go`.
